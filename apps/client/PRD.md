# Sisyphean PRD（产品需求文档）

Slogan: Copy the Newest, Paste Instantly.

Github: https://github.com/Hexi1997/Sisyphean

---

## 1. 项目名称与命名说明

**项目名**：Sisyphean

**命名原因**：

* 来源于希腊神话西西弗斯（Sisyphus）。
* “Sisyphean” 形容重复繁琐、无尽的任务，对应用户痛点：频繁打开下载文件夹、手动复制最新文件的重复操作。

---

## 2. 项目概述

Sisyphean 是一款轻量级系统级工具，帮助用户通过快捷键快速复制最新下载文件或用户指定目录的最新文件到系统剪贴板，实现零摩擦的文件操作体验。支持 Windows 和 macOS。

**核心痛点**：

* 用户下载文件后，必须打开下载文件夹才能复制。
* 系统未提供全局快捷键直接复制最新文件。
* 多浏览器、多下载目录情况下操作繁琐。
* 高频操作容易出错，效率低。

**解决方案**：

* 系统级全局快捷键触发复制最新文件。
* 支持多个下载路径，自动按时间获取最新文件。
* 后台常驻，开机启动确保快捷键随时可用。
* 可复制单个或多个最新文件。

---

## 3. 功能列表

| 功能         | 描述                                                                                          |
| ------------ | --------------------------------------------------------------------------------------------- |
| 全局快捷键   | 修饰键（mac系统默认：Option，win系统默认 Alt ) + Num，Num 表示复制文件数量，可自定修饰键组合 |
| 最新文件复制 | 自动获取所有下载路径中最新的文件并复制到系统剪贴板。                                          |
| 多路径支持   | 支持用户配置多个下载目录，自动汇总按时间排序。                                                |
| 后台常驻     | 程序后台运行，快捷键随时生效。可开机启动。                                                    |
| 多文件复制   | 支持复制最近 N 个文件，N 可通过快捷键 Num 参数控制。                                          |

---

## 4. 用户体验设计

* 快捷键触发后无需打开文件夹，立即复制最新文件。
* 系统托盘图标显示服务状态，可右键退出或暂停。
* 支持多浏览器、多下载目录统一处理。
* 错误或操作提示通过系统通知显示，提升反馈效率。
* GUI 配置界面，让用户方便添加路径、设置快捷键。

---

## 5. 跨平台策略

* Windows 与 macOS 默认下载路径不同，程序需自动获取并可配置。
* 系统级剪贴板操作根据平台 API 实现。
* 后台常驻与开机启动实现方式因平台而异。
* 用户配置文件格式统一（JSON/INI），便于跨平台迁移。

---

## 6. 品牌与传播策略

* 名称 Sisyphean 体现产品核心痛点——重复繁琐操作的解决方案。
* 强调文化感和效率感，让用户记住产品价值：快速、无摩擦、智能复制最新文件。
